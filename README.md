# logger4node [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url]
> Logger for Node.js, Logger anything you want.

## Installation

```sh
$ npm install --save logger4node
```

## Usage

```js
var logger = require('logger4node');

logger('Rainbow');
```
